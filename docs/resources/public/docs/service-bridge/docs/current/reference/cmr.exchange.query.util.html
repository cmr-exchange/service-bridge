subset->bounding-box

(subset->bounding-box elems)

In the CMR and EDSC, a bounding box is defined by the lower-left corner to the upper-right, furthermore, they defined this as a flattened list, ordering with longitude first. As such, a bounding box is of the form: [lower-longitude, lower-latitude, upper-longitude, upper-latitude].

This is the form that this function returns.

unique-params-keys

(unique-params-keys record-constructor)(unique-params-keys record-constructor other-allowed)

This function returns only the record fields that are unique to the record of the given style. This is done by checking against a hard-coded set of fields shared that have been declared as common to all other parameter styles (see the const namespace).
